# PetLinkID Repository Structure

## Root Directory Structure

```
petlinkid/
├── src/                      # React application source code
├── public/                   # Static assets
├── ios/                      # iOS native project (Capacitor)
├── supabase/                 # Backend functions and configuration
├── docs/                     # Documentation (you are here)
├── __tests__/                # Test files
├── scripts/                  # Build and maintenance scripts
├── capacitor.config.ts       # Capacitor development config (hot-reload)
├── capacitor.config.production.ts  # Capacitor production config (App Store)
├── vite.config.ts           # Vite build configuration
├── tailwind.config.ts       # Tailwind CSS configuration
└── package.json             # Node dependencies

```

## Key Configuration Files

### Build Configurations
- **`capacitor.config.ts`**: Development config with hot-reload from Lovable preview
- **`capacitor.config.production.ts`**: Production config for App Store builds (bundled assets)
- **`vite.config.ts`**: Vite bundler settings
- **`tailwind.config.ts`**: Design system and theme configuration

### iOS/Native
- **`ios/App/`**: Xcode project generated by Capacitor
  - `Info.plist`: iOS app metadata and permissions
  - `Assets.xcassets/`: App icons and launch images
  - `LaunchScreen.storyboard`: Launch screen UI

### Backend
- **`supabase/functions/`**: Edge functions (serverless backend)
- **`supabase/config.toml`**: Supabase project configuration
- **`supabase/migrations/`**: Database schema migrations

## Build Commands

```bash
# Web development
npm run dev

# iOS development (hot-reload)
npm run build
npx cap sync ios
npx cap open ios

# iOS production build
# 1. Comment out server block in capacitor.config.ts
# 2. Run:
VITE_BUILD_PROFILE=ios_free npm run build
npx cap sync ios
# 3. Open in Xcode and Archive for App Store

```

## Git Ignored Files

The following are generated/local and not tracked:
- `node_modules/` - npm packages
- `dist/` - build output
- `ios/App/Pods/` - iOS dependencies
- `ios/App/DerivedData/` - Xcode build cache
- `.env.local` - local environment overrides
- `*.log` - log files

See `.gitignore` for complete list.
